Hubble Eye

Hubble Eye is an instruming tool to monitor remote Computers/Servers resourses statuses and take necessary action by setting some rules.
Some of the example remote commands are "Task List", "End Task" and some of the example rules are by CPU or RAM monitoring and sending notifications.

Server Interface

How to use

First create a device entry in your account on the server and then download the setup wizard to install the service on the remote machine. Your device is ready to monitor afterwards.

Request task list

You can then see the tasks by their cpu usage and memory usage.

Request end task

You can request finishing a task.

Search tasks by name

You can check if certain task name is running on the client machine.

Get a live cpu/memory usage monitor

Shows a chart of cpu/memory usage for specific task.

Get a live overal cpu/memory usage

Shows a chart of cpu/memory usage for specific task.

Get and embeded code

So you can embed the HTML in you own site and monitor the server status.

Set you rules

Get notification when certain events happen.
